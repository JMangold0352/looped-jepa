## What is gitignored?

These are produced locally and referenced above but not committed:

- `checkpoints/`: trained weights (`.pt`)
- `runs/`: metrics logs (except you may commit selected JSON summaries)
- `data/`: CIFAR-10 download and transfer datasets

Committed in this repo: `results/**/*.md`, `results/**/*.json`, `visualizations/**/*.png`,
documentation, code, and configs.
